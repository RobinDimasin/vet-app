## Vetreatment: Pet Clinic
Live preview: [https://vetreatment.vercel.app/](https://vetreatment.vercel.app/)


## Run locally
Install Dependencies
```bash
npm install
# or
yarn install
```

Set up environment variables, see `.env.sample`

Start the server
```bash
npm run dev
# or
yarn dev
```
